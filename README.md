# ☢️ Pip-Boy Personal Portfolio

Bienvenido al terminal. Este repositorio contiene el código fuente de mi portfolio personal, diseñado con una estética retro-futurista inspirada en la interfaz del Pip-Boy de la saga Fallout.

## 🚀 Tecnologías Utilizadas
* **HTML5:** Estructura semántica del terminal.
* **CSS3:** Estilos personalizados, animaciones CRT, resplandor de fósforo verde y diseño responsive (Vanilla CSS).
* **JavaScript (Vanilla):** Lógica de la interfaz, efecto de máquina de escribir (typewriter), simulaciones de carga de datos y control de navegación.

## 🌟 Características Destacadas
* **Diseño Inmersivo:** Interfaz gráfica que simula un monitor de tubo (CRT) clásico, incluyendo "scanlines" (líneas de escaneo), aberración cromática y parpadeo de pantalla.
* **Efectos Dinámicos:** Animaciones progresivas de carga y revelación de texto letra por letra para ofrecer una auténtica experiencia retro.
* **Responsive Design:** Diseño adaptable para garantizar que el terminal sea funcional tanto en dispositivos móviles como en pantallas de escritorio.
* **Módulos del Sistema:**
  * `STATS`: Información del perfil profesional, nivel, habilidades y estadísticas extraídas de mi CV.
  * `PROJECTS`: Registro de misiones y desarrollos.
  * `LINKEDIN`: Acceso a la red profesional de la "Hermandad" a través de un código QR renderizado en el terminal.
  * `EMAIL`: Interfaz de comunicación simulada para establecer contacto directo.

## 🛠️ Instalación y Uso Local
Para iniciar este terminal en tu propia máquina de forma local:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/tu-repositorio.git
   ```
2. Accede a la carpeta del proyecto.
3. Abre el archivo `index.html` en cualquier navegador web moderno (no es estrictamente necesario un servidor local, aunque es recomendado para evitar problemas de CORS si se añaden recursos externos).
